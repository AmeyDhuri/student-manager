About

Student Manager is a simple web application built using Python and Flask that allows users to manage student records efficiently. The system includes authentication so users can securely register and log in before accessing the application.

The application provides a clean interface to add, edit, delete, search, and sort student records. It is designed as a beginner-friendly backend project to demonstrate CRUD operations, database integration, and basic web application structure.

This project was developed as part of my learning journey in backend development with Python, focusing on building real-world applications using Flask and SQLite.
